Curso de JavaScript

JavaScript (JS) é uma linguagem de programação que permite adicionar interatividade a sites e aplicativos. É uma linguagem de alto nível, legível por humanos, que é interpretada diretamente pelo navegador web. O JavaScript é usado para: Criar menus suspensos, formulários, galerias de fotos e gráficos animados. Desenvolver jogos, gráficos 2D e 3D animados. Criar aplicativos baseados em banco de dados.
